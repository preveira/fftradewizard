# FFTradeWizard

FFTradeWizard is a Fantasy Football Trade & Rest-of-Season Analyzer.

- Backend: Python + FastAPI
- Frontend: React + JavaScript (Vite)
- Docker: Containerization for both services

## Structure

```text
fftradewizard/
├── backend/   # FastAPI service
└── frontend/  # React app
```

## Getting Started

1. Start the backend (see `backend/README.md`).
2. Start the frontend (see `frontend/README.md`).
3. Open the frontend in your browser and start analyzing trades.
